Section 2: Set up your tools

Set up your tools

07 Install Chocolatey Package Manager (Windows Only)

Site: https://chocolatey.org/install

Make sure it works: choco

*Don’t forget to run choco commands as an administrator.

08 Install Hugo

https://gohugo.io/getting-started/installing

  • Windows: choco install hugo
  • macOS: brew install hugo
  • Ubuntu Linux: sudo apt-get install hugo

See the docs above for more complex installs.

09 Install Visual Studio Code and Extensions

https://code.visualstudio.com/

  • Windows: choco install visualstudiocode
  • Alternate/Manual: https://code.visualstudio.com/download

Extensions:

  • Code Spell Checker
  • Markdownlint
  • EditorConfig for VS Code
  • Better TOML

10 Installing Git

Windows: choco install git Windows (Manual): https://git-scm.com/downloads

Mac: brew install git Manual: https://git-scm.com/download/mac

Linux (Debian-based):

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install git

Linux (Red Hat-based):

sudo yum upgrade
sudo yum install git

11 Learning Git

GitLens: https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens

Basic Git commands:

  • git status
  • git add .
  • git commit -m “some message”
  • git clone
  • git push
  • git pull

Resources for learning Git: https://try.github.io/
